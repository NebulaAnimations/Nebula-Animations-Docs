Overlay Settings

Enable Overlay​

Toggle overlay on or off.

Image Overlay​

This section lets you add a image as an overlay texture.

Flipbook Overlay​

This section lets you add a flipbook as an overlay texture. Settings for flipbook textures will be found in the Flipbook Settings page.

Overlay Tint​

A color and intensity tint that works just like the one in Main Settings.

Edge Fade​

Adds an edge fade similar to the one in Main Settings.

Panning Settings​

Allows you to pan the overlay texture. (Image overlay only)

Flipbook Settings​

Use these settings to set up your flipbook texture if you are unsure how flipbooks work you can see a description here: https://wiki.amplify.pt/index.php?title=Unity_Products:Amplify_Shader_Editor/Flipbook_UV_Animation
